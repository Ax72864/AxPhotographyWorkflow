前端版本走站间直连 + 时刻表分段插值，路径不贴合真实铁路。
① 生成 schedule JSON：cd tools/railway && uv run python train2kml.py G1 2025-01-15 --schedule-only -o output/G1.json
② 选择该 JSON 文件 → 调整下方参数 → 点击"导入"。
如需路径吸附真实铁路线，请改用 --snap -o out.kml 生成 KML，再用左上角"加载 KML"导入。
粘贴由 schedule_fetcher.py / train2kml.py --schedule-only 输出的 JSON。
必须字段：depart_date（YYYY-MM-DD）+ stations[]，每站含 name、arrive（HH:MM，起点站为 null）、start（HH:MM，终点站为 null）、day（跨日数，默认 0）、lon、lat。 别名 arrive_time / depart_time / day_diff 也可识别。
